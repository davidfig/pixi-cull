new Simple (options)

simple.js:16

creates a simple cull

Name Type Description
options object optional
Name Type Default Description
visible boolean visible optional

parameter of the object to set (usually visible or renderable)
calculatePIXI boolean true optional

calculate pixi.js bounding box automatically; if this is set to false then it uses object[options.AABB] for bounding box
dirtyTest string true optional

only update spatial hash for objects with object[options.dirtyTest]=true; this has a HUGE impact on performance
AABB string AABB optional

object property that holds bounding box so that object[type] = { x: number, y: number, width: number, height: number }; not needed if options.calculatePIXI=true
