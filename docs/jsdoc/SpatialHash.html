new SpatialHash (options)

spatial-hash.js:20

creates a spatial-hash cull

Name Type Description
options object optional
Name Type Default Description
size number 1000 optional

cell size used to create hash (xSize = ySize)
xSize number optional

horizontal cell size
ySize number optional

vertical cell size
calculatePIXI boolean true optional

calculate bounding box automatically; if this is set to false then it uses object[options.AABB] for bounding box
visible boolean visible optional

parameter of the object to set (usually visible or renderable)
simpleTest boolean true optional

iterate through visible buckets to check for bounds
dirtyTest string true optional

only update spatial hash for objects with object[options.dirtyTest]=true; this has a HUGE impact on performance
AABB string AABB optional

object property that holds bounding box so that object[type] = { x: number, y: number, width: number, height: number }
spatial string spatial optional

object property that holds object's hash list
dirty string dirty optional

object property for dirtyTest
